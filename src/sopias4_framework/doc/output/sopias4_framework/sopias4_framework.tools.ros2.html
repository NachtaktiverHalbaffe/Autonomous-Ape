sopias4_framework.tools.ros2.costmap_tools module

This contains multiple helpful functions to interact with a PyCostmap2D which is used in the Python implementations of the Navigation2 plugins. Most of them a wrapper around the own functions of PyCostmap2 and only apply more appropriate types to work with. Also the built-in functions of PyCostmap2D can often be helpful too.
